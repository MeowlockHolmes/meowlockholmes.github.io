GRADUATION PROJECT

3D object retrieval using point clouds (PCs) and PC datasets



In this project we explored how can we classify 3D objects with using Unity AR.
Apart from the general methods followed in today's object classifications,
we aimed to do this with the Point Cloud, where new products are tried to be introduced.
With this project, we aimed to provide the user with the point cloud data of
the objects in our daily life and present objects that are close to them by classifying them
and then create and compare similar objects in the virtual environment according to their demands.

Made with Unity and Blender

404 Memory not found (Computer Graphics Lecture)


404 Memory not found is a 3 Dimensional, First person, closed world
puzzle game made with WebGL in 5 days without any library used
with 2 programmers and 1 modeller for Computer Graphichs Lecture
The player is stuck inside the building and they have lost their memories.
In order to regain their memories and to get out of the building
they have to collect certain objects.

Made with WebGL and Blender.
